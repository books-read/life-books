## 一、学校与社会：衡量标准不同

<span style="background: #fef08a; padding: 0 2px;">原因很简单：学校分数主要考记忆、理解和自控；社会则更看重想象力、情商、交际能力和态度等综合特质。</span>

现实社会有点像「足球经理」里的能力值表：技术、心理、体格等多维考量，但真实社会远比游戏复杂。


#### 真正进入社会，大家衡量一个人的标准是什么？<br/>
1、社会不会因学历或寒窗苦读认可你——那不会为他人带来收益。<br/>衡量标准归根结底只有一点：<span style="background: #fef08a; padding: 0 2px;">那就是你有没有为他人、为社会做出贡献。</span>他人用付费回应；不能让人付费，往往就没有价值。<br/>把命运掌握在自己手中，在竞争中学习竞争，在前进中树立新观念。

认清什么对社会有价值再去做，少做无用功。并非鼓吹「读书无用」，而是别为迎合他人、随大流只拼分数。先工作、发现不足再回炉读书，才是正确路径。

<span style="background: #fef08a; padding: 0 2px;">「三观」碎了，收拾心情，尽快踏上前路，有更多更新更好的事物等着你，这也是一种「碎碎平安」。</span>

## 七、工作中的困难与「幼稚儿」

工作中的困难往往比想象中多：设计好产品却建不成型、成型了推不出去、推广了转化不了、有转化又分析不了数据。

创业时最大的体会是：**<span style="background: #fef08a; padding: 0 2px;">这世界的「幼稚儿」太多。「幼稚儿」指年龄已成年、行为模式却仍是婴儿的人。</span>** <br/>
例如：
- 情侣吵架心情差就不上班、还不许扣工资的；
- 干得一塌糊涂还起早贪黑装努力、非要老板涨薪的；
- 非要一万五的椅子、出差必头等舱否则没心情的；
- 「宝宝不开心但宝宝不说」、要老板哄好才开工的；
- 「老板有钱为什么不分我、宝宝要哭了」的。

要在创业或事业上有所作为，**第一件事就是选择成人，真正成为一个成人。**

**真正的成年人只有一个标准**：<br/>
能否自由做出选择、是否明白自由选择的代价，并愿为选择承担责任。这是判断一个人、群体乃至国家是否成年、是否理性的标准。

一个人真正找到自己。

## 九、增量思维：做从未来看正确的事

**做从未来看正确的事，本质是关注增量而非存量。** 这个时代更看重你将带来什么，而不是你曾经拥有什么。

- 普通人关注存量（谁戴金表、拿爱马仕），婚姻选择上尤其明显。
- 聪明人与优秀企业家关注增长；马云录学生只看未来能给湖畔带来什么，不看有没有钱。
- 新时代企业关注你积累增量的方式；元认知即「认知的认知方法」，才是核心竞争力。
- 互联网公司招聘普遍看重这点，有增量思维的人更容易提升。

华尔街不是中国丈母娘，他们喜欢亚马逊那种象征未来的；资本市场看增量与增长，看的是你未来想成就什么、用什么方式达成。即便谈增量，视角也不同：有人看增长率，有人看超预期的增长率。人类自进入互联网时代，就被绑上一辆只看增量与增长的战车，永不停歇。

**不要被存量羁绊，紧盯增量，做从未来看正确的事。**

这个时代既残酷也温柔：成就与光荣一旦发生就没了价值；失败与困难一旦出现也变得无足轻重。

我在美国加入过 Ripple，在北京读过北大；它们与湖畔的共同点是：别人务实时务虚、别人谈实力时谈梦想、别人谈吃饱饭时聊改变世界。

北大给我「喷子」精神、不畏权威，宾大给我解决问题的能力；两种精神在我身上互补，宾大把我从空谈者打磨成实干家。<br/>
质疑权威是创新者必备，但不能为质疑而质疑，只有解决问题、获得满足，才是合格的企业家与创业者。我现有的商业理论、价值观、世界观，多来自马云。

## 十三、人生的「第一次」

**人生的第一次，很重要** <br/>
第一次最初始、最基础，对未来有强烈的塑造与指导意义，尤其体现在主动选择上；它不是短暂孤立的行为，而是与一生相伴、最初也最持续的战略。<span style="background: #fef08a; padding: 0 2px;">每一个第一次要尽力不将就、果断不犹豫。</span>

古语云：「国之大事，唯祀与戎。」<br/>
「第一次」的意义堪比人生的战争与祭祀：拥有什么、放弃什么、选择什么，会决定格局、气度与成长，如第一座城市、第一所大学、第一份工作、第一位导师、第一个国家、第一家公司等。<br/>

「种一棵树，最好的时间是十年前，其次是现在。」种树即人生各个重要的第一次；想明白是种树、种花还是种草，一旦决定就执行到底。<br/>
「第一次」不单是「一次」，更是最重要的一次；第一步的方向与时间，几乎决定未来道路。<br/>
想好了，就上路吧。

## 十五、这是个只看结果的世界

### 努力与结果

努力本身没有意义，意义在于呈现为对他人的价值与结果。**努力的廉价自我感动是绝大多数人无法赚钱的核心原因之一。** 在高度竞争的市场经济里，获得收入与实现价值的前提是为他人创造价值；结果对应你创造的特定价值，努力只是动机、态度与现象。

### 不对结果负责，就是不负责

创业越久越能感到：创始人若不能在内部建立好机制、树立对结果负责的意识，公司必将万劫不复。不对结果负责，评价标准就会「动机化」「无意义化」，挫伤真正能干者的积极性。世界进步需要指标来衡量，努力与廉价自我感动不在其中。

### 三步走，真正做到对结果负责

1. **第一步**：制定共同可见的目标及关键衡量指标（如 OKR）。
2. **第二步**：每月、每季度、每半年制定 OKR 计划，制定后保持绝对专注。
3. **第三步**：实行奖惩机制，有奖有罚、奖惩分明；奖励以创造的价值为基础，而非以能实施多大破坏为依据。

**努力，如果方式不对、结果不佳，就是无力。从现在开始，对结果负责，剔除杂念，保持专注，你就已经开始对自己负责了。**

## 十七、买房买车结婚养娃的悲剧

### 核心原因

绝大多数同龄人没法发家致富，不是因为能力、观念或起点差距，而是把 99% 时间用在「买房买车结婚、养房养车养娃」上——在你于互联网摸爬滚打之时，他们选择了另一条路。

### 悲剧所在

20～30 岁这十年，本是学习力、精力、魄力最强的阶段，却少有人把精力用于个人提升与改变社会，而是迅速妥协找一份稳定低薪工作，投入买房买车结婚养娃的传宗接代大业。这就是当代中国青年的一大悲剧。

### 租房的三大好处

- **成本低**
- **距离近**：可随工作地点随时切换
- **共享经济**：让租房变得简单、开心、实惠

## 十八、摒弃穷人思维

### 穷人思维

**穷人还是富人，核心是一种精神状态；穷人思维是一种单变量思维。** 拥有穷人思维的人用单一变量思考问题，例如把找伴侣简化为「没房子」，心里好受却失去了认真分析、继续奋斗、最终胜出的可能。

### 承认多变量，适应多变量，拥有富人思维

富人思维的核心是承认多变量、多维度竞争与变化速度，知道生态中的竞争优势是创新。是穷人还是富人，核心在思维：有富人思维，一贫如洗也前途远大；有穷人思维，腰缠万贯也毫无发展。**只要选对了路，走得慢一些，又有什么关系呢？**

## 二十三、财富自由的定义

### 定义

**财富自由就是：不需要为金钱做出价值观的妥协。** 妥协包括付出时间、金钱、注意力、情感和尊严。

### 是什么 / 不是什么

财富自由是财富独立带来的精神自由状态，与享乐、文艺追求无关；它不能带来真正的快乐（快乐来自创造，如创业与工作），只能让人免除因缺钱而产生的麻烦与妥协。

### 理解与态度

**对财富的正确理解，甚至发生在赚到钱之前。** 财富自由是一种精神理解状态，背后有复杂纵深；归根结底，是一种生活方式、精神信仰，值得捍卫的世界观与价值观。

## 二十四、杠杆：教育与期权的本质

无论创业者还是普通人，获得财富的方式一样：**通过杠杆实现核心个人资产的增值。**

- **教育的本质**：作为资产的「人」在不断升值。
- **期权的本质**：同理，也是资产增值的杠杆。

## 二十五、如何创业

### 创业三要

- **第一，定下来就是一辈子。** 创业是一辈子要干的事业。
- **第二，要坚持。** 员工跑了、伴侣走了……困难常有，即便如此也要坚持。
- **第三，要积累。** 团队与伙伴靠慢慢积累，创业是马拉松，不要急。

### 机会与准备

**夯实基础，等待机会；机会来时，用强大执行力抓住。** <br/>
机会没来，多想想是否没准备好；与其拿着放大镜找机会，不如先让自己变得更优秀。

## 结语

### 关于人生态度

为自己体面而自由地活着，是青春最有意义的方向；承认自己为自己而活，是思想上的成人礼。与自己相处，追求内心平和；与他人相处，追求世界和平。真正的快乐不来自他人认可，而来自对人生的掌控与自我成长的喜悦；未经自由选择的幸福，不值得过。

### 关于格局与选择

人生最重要的是格局，格局如同基因，从源头决定生长。心里有火光，就愿意向上。培养格局要看世界、见最优秀的人；格局要对齐天赋与志向，每一个第一次要尽力不将就、果断不犹豫。想好了，就上路吧。

### 关于社会与价值

社会衡量一个人的标准，归根结底是：你有没有为他人、为社会做出贡献；他人用付费回应。努力的意义在于呈现为对他人的价值与结果，而非廉价自我感动；对结果负责，才对自己负责。信契约、重态度；做从未来看正确的事，关注增量而非存量。这个时代既残酷也温柔——成就一旦发生就没了价值，失败与困难一旦出现也变得无足轻重。

### 关于财富与创业

财富自由是不必为金钱做价值观的妥协；获得财富的方式是通过杠杆实现核心个人资产的增值。年轻人向钱看、利己主义可以成为新道德，但要不断汲取营养、坚持学习。应许之地是互联网；在挣到第一个 100 万前，把多数时间精力用于提升自我、为进入互联网做准备。钱不稀缺，稀缺的是奋斗、闯劲与个人。创业要定下来一辈子、坚持、积累；夯实基础等机会，机会来时用执行力抓住。Problem Solver 是残酷世界最需要的；企业家核心竞争力是对问题的看法不断更新的能力。

### 一句收束

**这世界既残酷，也温柔。把命运掌握在自己手中，在竞争中学习竞争，在前进中树立新观念。**
